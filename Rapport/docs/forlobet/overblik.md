# H6 – overblik

Svendeprøven er den **afsluttende prøve**, du skal bestå for at få den faglige svendegrad. Prøven samler **praktiske færdigheder** og **teoretisk viden** inden for programmering – og relaterede områder – i ét sammenhængende forløb.

## Standard forløb

| | |
|--|--|
| **Varighed** | **25 arbejdsdage** (5 uger) |
| **Aflevering** | Senest på **dag 20** – se [tidsplan & Gantt](tidsplan.md) |
| **Eksamen** | I **uge 5**: typisk **1–2 dage** – se [Eksamen](eksamen.md) |

### Hvad skal du aflevere?

Ved aflevering forventes blandt andet:

- **Case** (projektoplæg / casebeskrivelse)
- **Logbog** med **estimeret og realiseret** tidsplan (fra dag 1)
- **Produktrapport** (dokumentation af produktet)
- **Procesrapport** (projektplanlægning med Gantt, logbog, konklusion — jf. MAGS-skabelon)
- **Kildekode** – typisk via **GitHub** (se [nyttige links](links.md))

!!! info "To separate rapporter"

    Lovgivningen skelner mellem **produktrapport** og **procesrapport**. Du finder skabeloner for begge under *Skabelon · Produkt* og *Skabelon · Proces* i menuen.
